# Olive UI Clone

A pixel-perfect recreation of the Olive landing page hero section, built using React (Vite) and Tailwind CSS.

## 🚀 Live Demo
https://olive-ui-clone.vercel.app

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS

## ✨ Features
- Responsive design
- Clean and modern UI
- Hover interactions
- Mobile UI mockup

## 📌 About the Project
This project was built as part of a UI assignment, focusing on accurately replicating a real-world SaaS landing page design with attention to detail.
